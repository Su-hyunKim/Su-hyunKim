** Positioning(위치) 속성 **

=> left, top, right, bottom
=> 위치값을 줄때에는 position 속성을 설정해야함.

** Position
=> 요소의 위치를 상대적으로 나타나게 할 것인지,
   또는 항상 같은 위치에 나타나게 할 것인지 설정.
=> 절대좌표 : 값에 의한 절대위치 (항상 같은 위치)
    - absolute : 다른요소에 무관하게 지정된 절대위치 단, 부모위치 기준
    - fixed : 항상 Body를 기준으로 함 (부모위치 와 무관)
=> 상대좌표 : 요소를 입력순서에 따른 상대 위치 지정
    - relative : 초기위치(Position을 지정하지 않았을때 본래 있어야 하는 위치)에서 이동
    			-> 부모위치를 시작점으로 이전요소들의 크기를 반영하고 margin을 계산하여 이동
    - static : 상대 위치 좌표
